# Brain Web Documentation

This directory contains the main documentation for the Brain Web project.

## Structure

- **Root docs/** - Active, living documentation that is currently relevant
- **docs/archive/** - Historical documentation, implementation summaries, and archived guides

## Key Documents

- `README.md` - Main project README
- `QUICKSTART.md` - Quick start guide
- `CODEBASE_OVERVIEW.md` - High-level codebase structure and architecture
- `NEO4J_SETUP.md` - Neo4j database setup instructions
- `MULTI_SOURCE_ARCHITECTURE.md` - Architecture documentation

## Archived Documents

The `archive/` folder contains:
- Phase implementation summaries
- Testing guides and examples
- Implementation verification guides
- Historical documentation

These are kept for reference but may not reflect the current state of the codebase.
